# salesOrder
Aplicación que conecta clientes, vendedores y administración para simplificar el flujo de pedidos. Los clientes realizan pedidos mediante un enlace del vendedor, el vendedor gestiona y valida, y la administración procesa.
Frontend: React-ts.
Backend: Nestjs.
DB: MySQL.
Testing: Jestjs
